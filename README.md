# Health-Flow
A Data-Driven Framework for  Behavioural Change in HSE Urgent Care

#What We Do
- Forecast hospital pressure (A&E demand proxy)
- Classify hospitals into Green / Amber / Red
- Provide personalised hospital recommendations
- Help users make informed decisions before attending A&E

#Technology Stack
- Streamlit
- Pandas
- Plotly
- SARIMAX forecasting (backend model)

#Purpose
Reduce A&E overcrowding by guiding patients to less pressured hospitals in real-time.
